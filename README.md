
## Installation

Install Django for the backend and Frontend
```bash
    pip install django
```

## Setup the Django
Setup the Django in the PC
```bash
    django-admin startproject mysite
```
## Required Libraries

Install the NLTK 
```bash
    pip install NLTK
```
Install pyaiml21
```bash
    pip install pyaiml21
```
Install Beautifulsoup
```bash
    pip install Beautifulsoup4
```
Install py2neo
```bash
    pip install py2neo
```

## Start the Server

``` bash
    python manage.py runserver
```
